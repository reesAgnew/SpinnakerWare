Error Reports

If a computer has generated an error report, there's a simple way to send that report to us.

Be sure you're in front of the computer that the error was reported on (each computer has their own file for reporting errors) and within OD Office Pro go to Tools>>Submit Error Report.

We'll get an email with the attachment. Please call or email separately, to help us better understand the situation in which this problem occurred.
